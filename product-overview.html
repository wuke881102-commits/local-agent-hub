本地 Agent 工作台

一个运行在本机的桌面工具。它通过飞书官方 CLI 读取你有权限访问的文档、知识库、多维表格、PDF、会议纪要，以及云盘上传的 Excel / Word / HTML 文件，在本地建立索引，再用内置的 AI Agent 处理这些内容。

简介

它做什么，不做什么

工作台不替代飞书，也不会把内容上传到外部服务器。它以你本人的飞书身份登录，只能读取你原本就能看到的内容。

权限与数据
索引、生成的草稿和日志都保存在你电脑的本地目录。涉及发送的操作（发消息、发邮件、写回飞书）需要你手动确认后才会执行，工具不会自动对外发送任何内容。
能力

八个内置 Agent

每个 Agent 负责一类任务，可以单独运行，也可以在任务场景里组合使用。

A1

HTML 页面生成

把飞书文档、或云盘上传的 Word / HTML 文件，整理成 HTML 页面、知识卡片或 FAQ。

A2

文档地图

列出可访问的飞书文档和表格，按类型、归属和重复情况归类。

A3

摘要标签回填

为文档批量生成摘要和标签，写入本地索引，也可以写回飞书。

A4

知识治理

找出重复、过期或缺少负责人的文档，并给出处理建议。

A5

多维表格分析

按表格数据归纳并出图——柱状、饼图、折线、甘特、架构图等，图表可下载。一次分析全部子表，也支持云盘上传的 Excel 文件。

A6

PDF 识别

识别云盘 PDF 的全文、字段和表格，支持逐页识别和合同金额测算。

A7

会议纪要

把妙记或会议记录整理成摘要、决策、待办和风险点。

A8

协作分发

生成消息、邮件、任务和日程提醒的草稿，确认后再发送。

支持的文件类型

能处理什么，怎么处理

工作台区分两类对象：飞书在线对象（文档、表格等，通过内容接口直接读）和云盘上传文件（.xlsx / .docx / .html / .pdf 等二进制文件，先下载到本地再解析）。下表列出每种类型走哪个场景、以什么方式处理。

类型来源场景 / Agent工作方式
飞书文档 docx · doc · wiki在线对象内容生产、知识库治理抽取正文（含内嵌图识别）→ 重组成 HTML 页面，或生成摘要 / 标签。
多维表格 bitable在线对象表格分析读取全部子表 → 列画像 → AI 规划并出图（柱状 / 饼 / 折线 / 甘特 / 架构图），数字由本地精确聚合。
电子表格 sheet在线对象表格分析同多维表格：逐工作表出图，并支持「问数据」自然语言查询。
演示文稿 slides在线对象内容生产抽取页面文字 → 重组成 HTML 页面。
会议 / 妙记 meeting在线对象会议沉淀整理成摘要、决策、待办和风险点。
PDF .pdf上传文件PDF 识别下载 → 逐页抽取正文与表格，扫描页走 OCR → 摘要 / 字段 / 合同金额测算。
Excel .xlsx · .xls上传文件表格分析下载 → 本地解析每个工作表 → 与在线表格相同的出图与「问数据」。
Word .docx上传文件内容生产下载 → 本地解析段落与表格 → 重组成 HTML 页面。
网页 .html · .htm上传文件内容生产下载 → 提取正文 → 重组成 HTML 页面。
为什么区分两类
飞书在线对象有内容接口，可直接读结构化内容；云盘上传文件只是存放的二进制，必须先下载到本机，再用对应解析器抽取（Excel→openpyxl，Word→python-docx，网页→BeautifulSoup，PDF→PyMuPDF）。解析全部在本地完成，不外传。在“本地资产”列表里，每个文件会按其类型直接显示可用的操作按钮。
使用流程

从安装到产出，五步

整个过程都在本地完成。

1

安装并启动

运行安装包，程序在系统托盘运行，并打开本地页面 127.0.0.1:8787。安装包内置了 Node 和 lark-cli，不需要另外配置环境。

2

授权飞书

程序内置专用应用 Feishu CLI。用设备码方式，以你本人的飞书身份完成一次授权登录。

3

刷新索引

拉取你能访问的文档、知识库、多维表格、云盘 PDF 和上传文件（Excel / Word / HTML）以及会议，建立本地 SQLite 索引。

4

运行 Agent

选一个任务场景或单个 Agent，对选定的对象运行。过程和结果在“运行记录”里可以查看。

5

查看产出

生成的内容保存为本地草稿。需要时再确认写回飞书。

本地架构，可离线分发。模型按任务类型分成几档，平衡速度、成本和质量。

数据在本地

索引、草稿和日志都写在本机 %LOCALAPPDATA% 目录下。

最小权限

只申请索引和读取需要的权限范围。发送类操作需要手动确认。

用你的身份

用专用应用 Feishu CLI，以你自己的飞书账号访问，能看到的内容和你登录飞书时一致。

开始使用

四步上手

1

安装

运行 LocalAgentHub-Setup.exe。首次安装可以勾选“重置本地数据”，从干净状态开始。

2

授权

打开工作台页面，以本人飞书身份对 Feishu CLI 完成一次授权。

3

建索引

点“刷新索引”，把可访问的飞书内容同步到本地。

4

运行

选一个任务场景或 Agent，运行并查看产出。
